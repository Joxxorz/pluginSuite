modalJocky

A simple plugin to enable use of modals in front end development.

Version: 1.0

Download javascript Download CSS

This plugin makes use of a link with a data-model attribute, targetting an already hidden modal with content inside further down the page.

You can view the source code for further clarification or contact me through the details on the contact tab if you need help.

Basic Usage

This is the default use of modalJocky, please see below how this can be implemented in jQuery and html.

Click here for demo 
            jQuery
            
            $('.whiteCross').click(function(){
              $(this).modalJocky();
            });
            
            HTML
            
            <a class="whiteCross" data-modal="white-cross">Demo White Cross Modal</a>
            <div style="display: none;" class="white-cross modalJocky">content here</div>

Different Close Button

This is a demo on setting a different close button. Anything similar (images, text etc.) can be used in the same way.

Click here for demo 
            jQuery
            
              $('.fontAwesome').click(function(){
                $(this).modalJocky({
                  closeBtn: '<i class="fa fa-times"></i>'
                });
              });
            
            HTML
            
            <a class="fontAwesome" data-modal="font-awesome">Demo White Cross Modal</a>
            <div style="display: none;" class="font-awesome modalJocky">content here</div>

tabJocky

A simple plugin to turn a list into tabs.

Version: 1.0

Download javascript Download CSS

This plugin makes use of a link with a data-tab attribute, which changes the tab based on the set data-tab, using active states to hide or display each tab.

You can view the source code for further clarification or contact me through the details on the contact tab if you need help.

Basic Usage

This is the default use of tabJocky, please see below how this can be implemented in jQuery and html.

The jQuery searchs for a class on your hidden tabs, until it finds one which matches the data-tab of the li that was clicked.

            jQuery
            
            $('.tab-selection .tab').click(function(){
              $(this).tabJocky();
            });
            
            HTML
            
            <ul class="tab-selection">
              <li class="tab active" data-tab="tab-jocky">tabJocky</li>
            </ul>
            <div class="tabJocky tab-jocky active">
            </div>

This tabbed area for the Jocky Suite is using this plugin in this format.

Multiple Tab Areas

This will be needed if you have multiple tab areas on one page, otherwise the jQuery will likely crash or show unexpected results.

You control which tab area is being selected with a class added to the tab area.

Click here for demo 
            jQuery
            
            $('.tab-selection .tab').click(function(){
              $(this).tabJocky({
                multitab: 'multitab1'
              });
            });
            
            HTML
            
            <ul class="tab-selection">
              <li class="tab active" data-tab="tab-jocky">tabJocky</li>
            </ul>
            <div class="tabJocky multitab1 tab-jocky active">
            </div>
